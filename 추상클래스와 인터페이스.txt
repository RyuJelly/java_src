InputStream -> 추상클래스
	-> read() : 추상메서드

FileInputStream, BufferedInputStream -> read() : 오버라이딩

추상클래스(인터페이스) 용도 : 코드의 독립성 유지, 코드 일관성 => 표준화(설계)


인터페이스
	-자바의 다중 상속을 대신
	- Interface 인터페이스명
	- Class 클래스 implements 인터페이스1, 인터페이스2
		=> 추상메서드 오버라이딩 (강제)
	- 인터페이스로 형변환 가능하다.
		=> 자바의 다형성을 구현

	역할
		자신이 인터페이스를 직접 구현 -> 코드의 독립성 유지, 코드 일관성 => 표준화(설계)
		API 인터페이스를 구현 => 직접적으로 해당 기술을 구현하기 위한 용도
